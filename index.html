¿Qué es HTML y para qué sirve?

HTML (HyperText Markup Language) es el lenguaje de marcado utilizado para estructurar el contenido de las páginas web. Define elementos como encabezados, párrafos, listas, tablas, imágenes y enlaces. En conjunto con CSS (estilos) y JavaScript (interactividad), HTML forma la base de las páginas web.

HTML se utiliza para:

  • Marcar la estructura semántica de un contenido (títulos, párrafos, listas, tablas, etc.).
  • Incorporar imágenes, videos y enlaces.
  • Definir formularios para recoger datos de usuarios.
  • Crear estructuras accesibles y compatibles con navegadores.

Tabla de códigos HTML básicos

Etiqueta Ejemplo Función
<h1> a <h6> <h1>Título principal</h1> Encabezados (tres niveles principales: h1-h6)
<p> <p>Párrafo de texto</p> Párrafos de texto
<a href="..."> <a href="https://example.com">Enlace</a> Enlaces a otras páginas
<img src="..." alt="..."> <img src="imagen.jpg" alt="Descripción"> Insertar imágenes
<ul> / <li> <ul><li>Elemento</li>...</ul> Listas no ordenadas
<ol> / <li> <ol><li>Elemento</li>...</ol> Listas ordenadas
<table> / <tr> / <td> <table><tr><td>Celda</td></tr></table> Tablas de datos
<div> / <span> <div>Contenido</div> Contenedores genéricos para agrupar elementos

Qué se puede hacer con HTML

  • Estructurar contenido textual y multimedia
  • Crear formularios para capturar datos
  • Insertar imágenes, videos y audios
  • Definir hipervínculos entre páginas
  • Construir tablas y listas para organizar información
